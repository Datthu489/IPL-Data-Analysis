# 🏏 IPL Data Analysis (2008–2024)
Personal Project | 2026

## Overview
Analyzed IPL match and player performance data using Python 
(Pandas, Matplotlib, Seaborn) to uncover key trends across 
17 seasons and 1,095 matches.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Power BI Desktop
- Jupyter Notebook

## Dataset
- matches.csv — included in this repo (1,095 match records)
- deliveries.csv — file too large to upload on GitHub.
  Download from Kaggle:
  https://www.kaggle.com/datasets/patrickb1912/ipl-complete-dataset-20082020

## Key Insights
- Mumbai Indians most successful team — 144 wins
- Virat Kohli all-time top scorer — 8,014 runs
- Yuzvendra Chahal top wicket taker — 205 wickets
- 64% captains choose to field after winning toss
- Winning toss has only 50% correlation with match win

## Files

- IPL_Data_Analysis.ipynb — Complete Python analysis
